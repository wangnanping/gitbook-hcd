### 使用 gitbook 创建 hcd 项目文档

- 安装 npm install gitbook -g

* 创建 npm install -g gitbook-cli （查看不需要创建）

* 执行 gitbook install

- 执行 gitbook serve

#### 目录
  
 SUMMARY.md 左边导航菜单

 pageList   说明文档页面
